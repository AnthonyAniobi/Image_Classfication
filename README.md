### Cifar 10

Using the cifar 10 image classification dataset using Convolutional nural networks

- Loss: Sparse Categorical Crossentropy
- Optimizer: Adam
- Metrics: Accuracy
- Learning Rate: 0.001
- Batch Size: 32

**Train loss:** 0.6438 , **Train accuracy:** 0.7757

**Test loss:** 0.8629 , **Test accuracy:** 0.7077

### Mnist
